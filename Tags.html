Important HTML tags


This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Things to add: Div, header, OL and UL < TR>
Tag Name Purpose
h1–h6 Headings Include a heading (levels 1–6)
"<"Title">" Title Includes title
"<"p">" paragraph Sepearates text into a paragraph
"<"a href">" hyperlink links to something else on the internets
"<"th">" creating table puts a table header
"<"tr">" table data puts table data

Inline Elements

Things to add: Span, blockqoute? add Li,
Tag Name Purpose Example Result
em emphasized make emphasized text <em>Big booty</em> Big booty
strong Strong bolds text <Strong> Big booty </Strong> Big Booty
<code> Code text makes text lineup and be equal distant <code> Big booty </code> Big Booty
<a> anchor makes hyperlink to image or link <a href="//https://learnenough.com/"> Learn enough
